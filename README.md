# pop-gestion/legal

Documents légaux publics des applications mobiles éditées par **Pop Gestion** (SARL).

🔗 Site publié : <https://pop-gestion.github.io/legal/>

## Structure

```
.
├── _config.yml              # Configuration Jekyll (theme + baseurl)
├── index.md                 # Landing page
├── macave/
│   └── index.md             # Politique de confidentialité Ma Cave
└── README.md                # Ce fichier
```

## Maintenance

- Le site est généré automatiquement par GitHub Pages à chaque push sur `main`.
- Le theme utilisé est `minima` (theme par défaut Jekyll, accessible, sobre).
- Toute mise à jour d'une politique doit incrémenter la date "Dernière mise à jour" en haut du document.

## Activer GitHub Pages (1× au setup)

Settings → Pages → Source = "Deploy from a branch" → Branch = `main` / `(root)` → Save.

URL active après ~1 minute : https://pop-gestion.github.io/legal/
